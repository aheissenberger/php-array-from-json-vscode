**from JSON to PHP Array:**

```json
{ "key": [1, "string", true, null] }
```

**from Javascript Object/Array to PHP Code:**

```Javascript
{
  key: [1, "string", null, , true]
}
```

**from Javascript Code to PHP Array:**

```Javascript
[...Array(10).keys()]
```

**from PHP Array to JSON:**

```php
["key"=>[1,"string",NULL,true]]
```

**from PHP Code return value to JSON:**

```php
range(0,9)
```
